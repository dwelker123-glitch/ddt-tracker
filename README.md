# DDT Tracker

Production-ready foundation for DDT tracking, protected calculations, historical snapshots, management schedule uploads, and trend reporting.

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## GitHub Pages

The app is configured for GitHub Pages through `.github/workflows/deploy.yml`. Push to `main`, then enable Pages from GitHub Actions in the repository settings if needed.
